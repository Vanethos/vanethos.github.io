TreeHouse Projects

Stormy Weather App

Weather App that retrieves information from Dark Sky API using GPS information to pinpoint the user's psoition, supporting different layouts for Tablets and Phones.

This project was created during the TreeHouse Android Techdegree, where the objective was to refactor an existing app, so that it would use fragments and supported different layouts for phones and tablets.

Technologies used: LocationManager, Shared Preferences, Permissions, ViewPager, Fragments.

  • GitHub

Movie Night

Movie Discovery app that lets the user either search for movies and TV programs or shows a list of possible movies and tv programs based on different parameters, such as release date, rating and genre.
This project was created during the TreeHouse Android Techdegree, where the objective was to use the The Movie DB API to query results for the "Discover" feature based on user's parameters.
Technologies used: Retrofit, RxJava, Picasso, CardViews, RecyclerView with Custom Adapters.
